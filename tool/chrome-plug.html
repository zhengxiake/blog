# 珍藏版Chrome插件送给你们，不仅是程序员必备

大家好，消失了几天我又满血复活归来了，最近这几天太忙了一直在加班工作，这不昨天又干到凌晨一点，今天早上七点就起来了，到现在还都没有休息，现在只剩半血了，不对应该说现在只能爬着走了，但是一想到几天没有更新文章了，还是强忍着上来更新，证明我还行。

以前啊，熬夜随随便便，现在随着年龄的增长真是不行了，一天都是懵的，所以说人呀趁年轻要努力提升自身的能力，提升认知，让自己有更大的竞争力，不然年龄大了你怎么和那些年轻人比，他们比你狠，比你努力，而你就半条命了。

好了废话不多说了，今天我们给大家推荐几个我自己一直珍藏的 Chrome 插件，打造一个牛逼，炫酷不一样的浏览器，瞬间让你上一个档次，证明我们就是不一样。

# 1. Momentum

注意这个就牛逼了，它直接让你的 chrome 看上去就和别人不一样，简直是 chrome 的必备，用它我们可以创建一个炫酷的新标签页，而且每天都不一样，每天都送你一个高质量的背景，每一天都是不一样的心情。不信我们来看看？

酷不酷我就不说了，你们自己觉得，关键是每一天都不一样，而且还可以添加 todoList 更多功能你们可以自行探索，因为下面还有更重要的推荐给你们，这才是刚开始，你们就激动的不行了。

地址我都给你准备好了，直接传送门就可以安装了

Momentum：https://chrome.google.com/webstore/detail/momentum/laookkfknpbbblfpciffpaejjkokdgca

就是这么贴心。

# 2. New Tab Dribbble

这个就更厉害了，可以说是设计师的必备，你每次打开新的标签页都会第一时间把每日精品设计作品展示出来，你每天都能看到来自大神的作品，简直是灵感的源泉，作品来自 Dribbble。不过可惜的是它和 Momentum 两者只能同时存在一个。

New Tab Dribbble：https://chrome.google.com/webstore/detail/new-tab-dribbble/omhhjicbfncielplfgggjedgnkjoicjj

# 3. OneTab

OneTab 对我们技术人员来讲真是犹如神助啊，我们在工作中难免在开发的过程中查阅资料，打开很多标签页但是又舍不得删除，最主要的是浏览器的标签页打开了来以后不仅影响我们查找，还非常占用电脑的内存，是我们的电脑非常卡，以至于怀疑自己的电脑性能不好。

OneTab：https://chrome.google.com/webstore/detail/onetab/chphlpgkkbolifaimnlloiipkdnihall

有了 OneTab 这些根本就不是问题了，它可以帮你把所有标签页都收集到一个标签里，而且还能释放我们的内存，当我们需要的时候直接点击恢复我们继续使用。就是这么简单，粗暴。

# 4. Isometric Contributions

上面推荐了一些设计师，或者任何人都可以使用的工具那么这个应该就是程序员专属了，它可以让我们 Github 的贡献图变成 3D 效果，非常适合用来装逼有没有，是不是很高大上。

Isometric Contributions：https://chrome.google.com/webstore/detail/isometric-contributions/mjoedlfflcchnleknnceiplgaeoegien

# 5. GITHUBER

开发者的新标签页，它会创建一个新的标签页展示当前最优秀的开源项目，真是简直了，从此妈妈再也不用担心我学习写代码了，还支持谷歌、百度、必应、雅虎。

GITHUBER：https://chrome.google.com/webstore/detail/githuber-%E5%BC%80%E5%8F%91%E8%80%85%E7%9A%84%E6%96%B0%E6%A0%87%E7%AD%BE%E9%A1%B5/janmcneaglgklfljjcpihkkomeghljnf
