# 聪明的人都是乐观、懂规则的，而不是抱怨

今天打破以往的常态，我们不聊技术，我们聊聊认识上的事情，从一开始写公众号开始，我就想除了技术以外，平时也写一些别的东西，今天就聊一聊认识。

阶级固化，这个词频繁的出现在我们的视野里，它让我们感受到了世界的三六九等，让我们感受到富人越富穷人越穷。

也有很多人，感受到世界的不公平，为什么别人的条件总是那么优越，而自己却一无所有，不管自己怎么努力都不能达到别人的高度，别人却轻而易举的获取成功。

说到努力，十年寒窗让人最有感触，还记得之前有个北大的文科状元的说的一番话被刷屏了，他说我能享受到的资源与教育是农村的孩子永远也享受不到的，我接触的环境也是他们无法触及的，所以我比他们更容易获取成功，虽然这个例子已经被我们说烂了，可能大家看到都没有什么感觉了，但是不得不承认，他说的很对，寒门再难出贵子。

然而，有些人还停留着，十年寒窗就能够改变命名的认知里，我只能说咱们还是太年轻。

成功需要很多因素决定的，十年寒窗只是影响因素之一，和你的家境好，长得美，智商高等等都是因素之一，如果你仅仅以为你十年寒窗上了大学就算成功了，你就大错特错了！

大学才只是刚刚开始，但是有些人以为是结束了。

有些时候这也不怪我们自己，就拿我来说吧，我上高中的时候，我们班主任每天就给我们灌输一个思想，只要努力三年，你上了大学就轻松了，想怎么玩就怎么玩！

嗨...你猜最后怎么着，我还真真的就玩到整个大学毕业，当我意识到的时候，显然已经落后的别人一大截，与其说别人的错，不如说自己认知低。不过，只要我们意识到这些问题，行动起来，什么时候都不算晚。

其实人生是一场排位赛，玩过英雄联盟的同学都知道，为了体现玩家水平的高低，我们划分为很多个阶段，比如：青铜，白银，黄金，白金，钻石，王者。有人从青铜下降白银，就有人从白银下降为青铜，有人从钻石晋升为王者，就会有人从王者坠落到钻石，游戏都尚且如此，更何况我们的社会呢？

事实上，在任何国家或在任何的社会中都会分层次的，既然分层就会有些人上升，有些人会下降，从上到下的人我们称之为 “逆袭”，而现如今中国的社会就处于一个最好的时代，因为互联网的出现，到处都是机会，我们也会不断听到屌丝逆袭的故事。

从下到上总是辛苦的，而且我们要付出比别人更多的努力，因为不同的人存在不同的层次里，享受的资源，人脉，环境，家庭都会千差万别，如果有个条件比你好的人，你和他付出一样多的努力，你永远都不可能超越他，你只有付出比他要更多的努力才能够超越，然而我们很多人都有个误区就是只看到自己的努力，却没有看到别人的努力。

甚至即使你付出了 100% 的努力，但是别人只付出 10% 的努力，但是产生的结果也会比你好，所以我们看到的屌丝逆袭的人总是少数。

庆幸的是，并不是每个人都是这么努力，绝大部分人的一生都是处于松懈状态，这才给了我们逆袭的机会，只要我们一直努力下去，而且努力真的是最简单的事情。

最后，说了这么多我不是为了打击各位，相反的我是想让大家认清自身的情况，只有做到对自身的充分了解，才能意识到自己的不足，才能够去弥补，而不是还永远停留在大家都在统一的起跑线上。

最最后，努力是一段很长的路要走，我们并不需要追求从最底层上升到最高层，相反的只要我们能够通过努力上升几个台阶，就会活的很好。

最最最后，这个世界并没有欠你什么，我们需要做的就是找到自身的不足，做出足够的努力，然后去一步一步的改变自己，而不是整天抱怨社会的不公。
