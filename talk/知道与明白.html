# 「知道」与「明白」中间到底隔着怎么样的道路？

前段时间，被达克效应刷屏了，几乎每个人都分享了那张图片，人都是从愚昧之巅到绝望之谷，但是，能不能跑上开悟之坡真的是看每个人的造化了。

毕竟向上的路充满了荆棘与艰辛，向上生长是一个痛苦的过程。

先撇开能不能从绝望之谷到开悟之坡，单从愚昧之巅到绝望之谷都不是一件容易的事情。

为什么？一来认知有限，二来是自我感觉良好，所以完全是，不知道自己不知道，妥妥的站在愚昧的最高峰。

1.因为认知有限，明白其实很难

我是一个地地道道的农村人，而且家庭并不富裕，父母也没有什么文凭，完全的穷乡僻壤，就是这样的一个环境中长大。

小的时候，父母经常教育我说好好上学，上好学了就不用在家种地了，上学的时候能享受的资源也非常有限，以至于别人说他们小学就学英语课程我都会很诧异，因为我的小学中只有两门课，语文、数学除此之外别无。

周围同龄的小伙们一个一个的相继辍学，只有那么一两个在坚持着所谓的求学道路，因此当时也被人是看作是好好学生，其实呢？并不是，只是因为你就那两个上学的其中一个而已。

经常被老师教育知识可以改变命运，也知道农村人好像也只能通过上学去改变自己，冲破枷锁。

但是，当时自己看到的也好，自己的认知也好，貌似没有知识的人过的也没有那么糟糕。

因为你看不到，所以想不到，自己也不会想去得到，但是当自己去经历过这些之后才明白了其中的道理。

随着自己认知的升级与阅历的增加，以及自己的感同身受，才开始从知道到明白的过渡，这些其实都是受环境，条件，基础的不同而产生不同的结果，要不你看往往都是家庭越优秀的人成绩越好。

其实很多道理大家都知道，也知道一件事情的重要性，比如读书，写作，但是能明白读书，写作重要性的真的没有几个，因为如果他明白了一定会去做。

2.自我感觉良好，高估自己

毕业这么多年做过很多份工作，也是几经曲折最终走到了技术发展的道路上，进入互联网的第一份工作是做 UI 设计，最后因为看开发的工资高，刚好有一个机会所以转到了前端岗位。

大概在做了一年多的前端之后，有点自我感觉良好，平时遇到的问题也都能够解决，也经常看技术书籍提升自己的技术，当我以为这些已经足够的时候，所以我就出去用面试测试了自己的段位。

不测不知道，一测吓一跳啊，原来我一直停留在自己的世界里啊，无论从技术方面还是从沟通方面，还有很大的空间需要提升，一下子把我推到了谷底。

基于这次的失败，我重新审视了自己，然后爬起来继续向前走，利用一切能够利用的时间去学习提升自己，也放弃了很多的娱乐活动，现在几乎没有任何的娱乐时间了。

从「知道」到「明白」之后，即使每天感觉很累，但是似乎内心更加的坚定了。

「知道」到「明白」说难也不难，说容易也很不容易，苏格拉底说：未经审视的人生不值得度过。希望每个人都能审视出自己的道路。
